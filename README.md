# DSA210 Term Project
## Economic Spillover Effects of Wars on Neighboring Countries

This project examines whether recent wars create measurable economic spillover effects on neighboring or nearby host countries through refugee inflows and related macroeconomic changes.

## Project Proposal
[Project Proposal PDF](proposal.pdf)

## Scope
The project focuses on three conflicts:
- Syrian Civil War
- Russia–Ukraine War
- Afghanistan conflict

The host countries include Turkey, Lebanon, Jordan, Poland, Romania, Hungary, Slovakia, Moldova, Pakistan, and Iran.

## Data Sources
- UCDP Battle-Related Deaths Dataset
- UNHCR Refugee Data Finder
- World Bank World Development Indicators

## Planned Method
The project will use annual country-level data, combine conflict intensity, refugee exposure, and macroeconomic indicators, and begin with data collection, exploratory data analysis, and hypothesis testing.
